# windows_class_work
我的大三上课程设计——智慧窗帘系统

题目：智慧窗帘控制系统设计

设计目标：设计支持光照自动调节、定时开合的电动窗帘控制器。

功能要求细化：
电机驱动：驱动步进电机或直流减速电机，行程时间20~30秒（窗帘开合一次）
位置检测：通过霍尔传感器或光耦计步，定位精度±1cm

控制模式：
1、光控模式：光照>设定值自动关闭，<设定值自动打开；
2、时控模式：RTC设置2组定时（如07:00开/22:00关）；
3、手动模式：按键点动控制；
4、远程模式：手机APP开合控制，支持半开比例（0~100%）

人机交互：OLED显示当前开度百分比、光照值、模式状态；两个按键（开/停/关）；蜂鸣器提示音（开/关到位"滴"一声）
通信接口：HC06蓝牙实现远程控制
